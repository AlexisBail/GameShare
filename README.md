🎮 GameShare - Le jeu sans limite
GameShare est une plateforme web de location de consoles et de jeux vidéo (PS5, Xbox, Switch, VR, Retro). Ce projet a été réalisé dans le cadre de ma formation DWWM pour valider les compétences de conception, de maquetage et d'intégration d'interfaces utilisateur.

🚀 Le Projet
L'idée est de proposer une solution simple et accessible pour profiter des dernières nouveautés gaming ou organiser des événements sans avoir à acheter le matériel.

Fonctionnalités clés :

🏠 Accueil : Présentation du concept et des services.

📂 Catalogue : Liste des consoles et jeux disponibles avec système de filtrage.

🛒 Panier : Gestion des articles sélectionnés avant la location.

💳 Tunnel de commande : Parcours complet de la saisie des infos au paiement.

✅ Confirmation : Validation finale de la commande avec récapitulatif.

🛠️ Outils & Technologies
Design (Maquettage)
Figma : Conception de l'identité visuelle, du logo et des prototypes UI/UX.

Développement Front-end
HTML5 : Structure sémantique (SEO & Accessibilité).

CSS3 : Mise en page moderne avec Flexbox et CSS Grid.

Responsive Design : Interface totalement adaptée aux mobiles et tablettes.

Vite : Environnement de développement rapide.

JSON : Gestion des données produits (consoles et jeux).

Gestion de projet
Git / GitHub : Versioning du code et suivi du déploiement.

VS Code : Éditeur de texte principal.
